# MVP User Flow

Open Inventory Dashboard
→ View Today’s Inventory Summary
→ Select an Alert Category
→ Review Flagged Products
→ Open a Product for Details
→ Review the Recommended Action
→ Mark the Product as Reviewed

The MVP flow ends after the product is marked as reviewed. The Inventory Coordinator completes any operational action, such as placing an order, through the company’s existing process. StockWatch does not place orders, contact suppliers, or generate reports.
